Standardisasi Malvern

Validasi Instrumen & Pembuatan SOP Tahun 2023

Pada tahun 2023, laboratorium beralih dari penggunaan alat Robot Sifter konvensional ke instrumen analitik modern, Malvern Particle Size Analyzer.

  • Conditioning Alat: Instalasi awal dan konfigurasi parameter sistem.
  • Komparasi Metode: Uji sampel paralel antara Robot Sifter lama dan Malvern baru.
  • Penyusunan SOP: Membakukan parameter optimal ke dalam dokumen SOP resmi.
