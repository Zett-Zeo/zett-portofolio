LIDS: SDLC & Evolusi Sistem

Perjalanan transformasi digital dari modifikasi makro Excel sederhana menjadi ekosistem web mandiri terintegrasi.

Fase 1: Standardisasi & Visualisasi

  • Menormalisasi file pelaporan (Hatto Hiyari) yang terfragmentasi tahunan menjadi satu Single Source of Truth berbasis database berkelanjutan.
  • Mengembangkan dashboard pelaporan interaktif memanfaatkan Excel VBA Macro.
  • Mengimplementasikan automasi filter waktu dan kalkulasi statistik otomatis yang disajikan via visualisasi grafik (Bar & Donut chart).

Fase 2: Skalabilitas Cloud & Sinkronisasi

  • Merespons kendala limitasi perangkat keras (antrean input PC) dengan bermigrasi ke ekosistem Google Workspace (Forms & Sheets).
  • Membangun automasi data routing skala lab menggunakan Google Apps Script.
  • Menyatukan tiga aliran data krusial (Hatto Hiyari, KYT, dan Trouble Report) ke dalam satu Centralized Dashboard analitik.

Fase 3: Arsitektur Plugin & On-Premise Server

  • Mengevaluasi UI mobile yang suboptimal dan menekan pembatasan aplikasi pihak ketiga dengan merancang sistem Full Custom Web secara mandiri (PHP, JS, Tailwind) beserta database relasional PostgreSQL.
  • Berinovasi menciptakan wadah Arsitektur Plugin untuk Form Builder, Report Bank, My Report, dan Approval Workflow, sehingga aplikasi baru dapat dimuat (*load*) tanpa merombak sistem induk.
  • Mengeksekusi peran SysAdmin/DevOps: menyulap PC bekas menjadi lokal server (On-Premise) dengan OS Linux Mint dan web server Apache2.

Platform E-Learning Privat

Pengembangan Sistem Manajemen Pembelajaran (LMS) Berbasis Web untuk Klien Eksternal.

Selain mengembangkan sistem internal laboratorium, saya juga menangani proyek dari klien eksternal. Proyek ini adalah sebuah platform web mandiri yang dirancang khusus untuk memfasilitasi kegiatan belajar mengajar seorang guru privat.
